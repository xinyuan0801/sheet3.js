Sheet3.js

API

Getting Started

the script needed will be 

        
          <script defer src="sheet3.js" type="text/javascript"></script>
          <link rel="stylesheet" href="sheet3.css" />

you would also need jquery, the version used is 3.5.1

to start using sheet3.js, first initialize a sheet3 object 

      
        const NumberTest = [
          { first_name: "Leonard", last_name: "Faulconer", age: 45 },
          { first_name: "Editha", last_name: "Keogh", age: 49 },
          { first_name: "Jeth", last_name: "Hasson", age: 83 },
          { first_name: "Liz", last_name: "De Bruyn", age: 46 },
          { first_name: "Valentia", last_name: "Lasslett", age: 40 },
        ];
        var t1 = new sheet3({
          id: "t1",
          row: 3,
          col: 5,
          type: "readonly",
          location: "T1",
          style: "DM",
          mode: "vertical",
          data: NumberTest
        });

Options

all the options needed to initialize a sheet3 object

self.createTable()

will create table according to the options given when create the sheet3 object.

Note: all methods below can only be called after createTable() method is called

self.fillTable(inputData)

will fill the array of JSON data, inputData into the table created.

self.index(row, col)

this method will return the value in the cell in row number row and column number col given.

self.removeRow(row)

this method will remove the row at row number row

self.sortByCol(col, mode)

this method will sort table by the value in the column number col, in logic with the mode

self.addRow(JSONtoAdd, row)

this method will add the row represented by JSONtoAdd into index row in table row

self.search(headername, searchContent)

this method will return the tr element of the row with the value searchContent in the column of headername

self.fillHorAndVer(Vh, Hh, inputData)

this method will fill data into the table with mode HorAndVer with inputData

self.exportToJSON()

this method will return an array of JSON object represented the content in the table
note:only for horzontial headers table and vertical headers table

self.showRow(start, end)

this method will show the rows in table with row index from start to end and hide the rest

note:only for horzontial headers table and vertical headers table

self.resetShow()

this method will show all the rows that is hide when use showRow method.

note:only for horzontial headers table and vertical headers table

self.sum(col)

this method will return the sum of value in column with column index col

note:only for horzontial headers table

self.avg(col)

this method will return the avgerage of the column with given column index col

note:only for horzontial headers table

self.setCell(row ,col, val, isElement)

this method will set the cell with row number row and column number col with content col
this method support val in a form of html element, but it only works with readonly table

note:only for horzontial headers table and vertical headers table
